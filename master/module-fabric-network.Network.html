Interface: Network

fabric-network. Network

A Network represents the set of peers in a Fabric network. Applications should get a Network instance using the gateway's getNetwork method.

Methods

addBlockListener(listener [, options])

Add a listener to receive block events for this network. Blocks will be received in order and without duplication.
Parameters:
Name Type Argument Description
listener module:fabric-network.Network~BlockListener A block listener callback function.
options module:fabric-network.Network~ListenerOptions <optional>
 Listener options.
Returns:
The added listener.
Type
module:fabric-network.Network~BlockListener
Example
const listener: BlockListener = async (event) => {
    // Handle block event

    // Listener may remove itself if desired
    if (event.blockNumber === endBlock) {
        network.removeBlockListener(listener);
    }
}
const options: ListenerOptions = {
    startBlock: 1
};
await network.addBlockListener(listener, options);

Type
module:fabric-network.Network~CommitListener
Example

        // Handle peer communication error
    } else {
        // Handle transaction commit event
    }
}
const peers = network.channel.getEndorsers();
await network.addCommitListener(listener, peers, transactionId);
